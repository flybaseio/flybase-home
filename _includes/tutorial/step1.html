2. Now, let's install Flybase

To get you started, lets create a blank HTML page, and install the Flybase JavaScript client by adding the following line into the <head> section: 


	<script src="https://cdn.flybase.io/flybase.js"></script>

So it should look like this:


<html>
<head>
<script src="https://cdn.flybase.io/flybase.js"></script>
</head>
<body>

</body>
</html>
